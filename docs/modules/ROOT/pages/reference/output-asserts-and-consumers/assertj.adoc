then access them via `CommandResult.stdout().lines()` or `CommandResult.stdout().bytes()` after execution:

You can also assert on the raw bytes:

[source,java]
----
include::example$j21/AssertJStdoutTest.java[tag=imports]

include::example$j21/AssertJStdoutTest.java[tag=bytes,indent=0]
----

[WARNING]
====
Using `captureAll()`, `lines()` and `bytes()` API methods may cause high memory consumption
depending on the amount of data produced on `stdout` or `stderr`.
====
